# Yogesh Kumar Portfolio

A single-page, responsive portfolio site for Yogesh Kumar, highlighting product and data platform leadership experience.

## Features
- Clean navigation with responsive toggle for mobile screens.
- Sections for About, Experience, Skills, Education, Community work, and Contact details.
- Call-to-action banner for role availability and quick access to resume and email links.

## Structure
- `index.html` – Core page markup and content sections.
- `css/style.css` – Styling for layout, typography, and responsive behavior.
- `js/main.js` – Navigation toggle logic and smooth scrolling.

## Getting Started
1. Clone the repository and install any optional Live Server extension (or use a simple HTTP server).
2. Start a static server from the project root, e.g.:
   ```bash
   python -m http.server 8000
   ```
3. Visit `http://localhost:8000` in your browser to view the site.

## Customization
Edit the HTML content or adjust colors, spacing, and typography in `css/style.css` to tailor the portfolio to another profile.
